About the Commander Application

What is this application?

Commander is a web application that helps users keep track of the commands they use in their project.

Users can sort and filter through their commands

Users can also create, update and delete their commands



What I used to create this application

The frontend uses Angular 9 with Angular Material.

The backend uses ASP.NET Core WebApi 3.1, Entity Framework Core and Automapper.

The web api utilizes authentication via Microsoft Identity.
